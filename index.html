HTML

  • HTML is the foundation or framework of building a house, but for a website.
  • In HTML we use tags to tell the computer what format we want our elements in.
  • The tags hold the elements and the elements hold the content for the website.
  • The head element contains information about the webpage.
  • The body element represents the visible content that is shown to users.
  • HTML provides structure and content on a website, something for users to interact with.

CSS

  • CSS is for styling your website
  • There are 3 types of CSS; inline CSS, internal CSS, and external CSS.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the inside of an element.
  • You must link your CSS stylesheet to your html for it to load on your webpage.

Git

  • git status: show modified files in working directory.
  • git branch: lists branches.
  • git checkout main: switch to main branch.
  • git stash: temporarily save modified changes.
  • git log: show the commit history for active branch.
  • git clone [url]: clone a repository.

JavaScript

  • JavaScript is used for functionality of a website.
  • Functions are the building blocks of JavaScript.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code n a script.
